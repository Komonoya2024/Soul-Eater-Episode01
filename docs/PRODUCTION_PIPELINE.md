# Production Pipeline

## Phase 1 — Story Intake
匯入原作與改編資料，建立摘要、時間線、人物關係、世界規則、衝突節點及來源權利紀錄。

## Phase 2 — Agent Pipeline Configuration
設定 Director、Character、Environment、Prompt、Video、Assembly、QA 七類 Agent，統一輸入輸出契約與連續性規則。

## Phase 3 — Batch A Execution
範圍為 SH001–SH030：鏡頭卡鎖定 → Prompt Bundle → 候選影片 → 初審 → 初剪 → QA。

## Phase 4 — Batch B–D
- Batch B：SH031–SH060
- Batch C：SH061–SH090
- Batch D：SH091–SH120

## Stage Gates
每批次須通過：敘事、角色、環境、攝影、畫質、音訊、時長、權利與交付規格檢查，才可進入下一批次。
