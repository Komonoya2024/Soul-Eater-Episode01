# CHAR-001｜Expression Library

## 表情強度

- E0：完全中性
- E1：微弱情緒
- E2：明確但克制
- E3：強烈情緒
- E4：極端或崩潰

## 表情組

| ID | 表情 | 強度 | 使用情境 |
|---|---|---:|---|
| NY-E01 | 冷靜觀察 | E1 | 調查、潛伏 |
| NY-E02 | 輕度懷疑 | E1 | 發現矛盾 |
| NY-E03 | 壓抑憤怒 | E2 | 同伴受傷或遭欺騙 |
| NY-E04 | 瞬間恐懼 | E2 | 真相突然浮現 |
| NY-E05 | 戰鬥專注 | E2 | 交火、追逐、近身衝突 |
| NY-E06 | 悲痛克制 | E2 | 面對死亡或失敗 |
| NY-E07 | 意識混亂 | E3 | 融合干擾、記憶污染 |
| NY-E08 | 決絕 | E3 | 最終行動或犧牲選擇 |

## 生成要求

- 每種表情須保留相同臉部結構
- 表情變化應集中於眉眼、嘴角與下頜肌肉
- 禁止誇張動漫式表情
- 禁止因憤怒或悲痛導致臉型改變
- 所有表情先以無墨鏡版本校驗，再補 AR 墨鏡版本
