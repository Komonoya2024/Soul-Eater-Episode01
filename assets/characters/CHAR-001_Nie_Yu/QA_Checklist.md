# CHAR-001｜QA Checklist

## Identity

- [ ] 臉型與核准 Face Reference 一致
- [ ] 左眉尾淡疤位置一致
- [ ] 眼距、鼻寬、嘴角及下頜線一致
- [ ] 視覺年齡維持約 32 歲
- [ ] 髮型、髮際線及髮色一致

## Body

- [ ] 身高比例約 181 cm
- [ ] 約 7.6 頭身
- [ ] 精瘦實戰型體格
- [ ] 無過度健美或四肢比例漂移

## Costume and Props

- [ ] Costume ID 與 Shot Card 一致
- [ ] AR 墨鏡狀態正確
- [ ] 腰間槍套固定於右側
- [ ] 無未授權品牌標誌
- [ ] 連續場次無服裝跳接

## Expression and Pose

- [ ] Expression ID 正確
- [ ] 情緒強度符合場景
- [ ] Pose ID 正確
- [ ] 動作重心與接地自然
- [ ] 手指、手腕及眼鏡無變形

## Image Quality

- [ ] 寫實皮膚質感
- [ ] 無過度磨皮或年輕化
- [ ] 無動漫或插畫化傾向
- [ ] 無臉部互換
- [ ] 無多餘肢體或重複人物
- [ ] 無文字、Logo 或水印
